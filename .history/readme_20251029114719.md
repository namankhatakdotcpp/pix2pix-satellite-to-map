# 🛰️ Pix2Pix — Satellite → Map

A small demo project that converts satellite images into map-style renderings using the Pix2Pix conditional GAN (Isola et al., 2017). Trained on paired satellite↔map images, the model learns a one-to-one mapping using a U‑Net generator and a PatchGAN discriminator.

## Quick start

1. Create a virtual environment (macOS):
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. Run a quick demo (1 epoch):
   ```bash
   python src/train.py --epochs 1
   ```

---

## Training

Use src/train.py to train on your dataset. Common flags:
- --epochs: number of epochs (default in script)
- --batch-size: training batch size
- --dataset-dir: path to paired images

Example:
```bash
python src/train.py --dataset-dir data/ --epochs 50 --batch-size 8
```

Check outputs/ for saved example images and demo.gif. Save/load model weights under saved_models/.

---

## Model overview

- Generator: U‑Net (encoder → decoder with skip connections)  
- Discriminator: PatchGAN (patch-level classifier)  
- Losses:
  - Adversarial loss (GAN)
  - L1 reconstruction loss (encourages pixel-level similarity)
