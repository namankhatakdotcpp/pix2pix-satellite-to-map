# 🛰️ Pix2Pix Satellite → Map Translation using GANs

This project demonstrates how **Conditional GANs (Pix2Pix)** can convert **satellite images** into **map renderings**.  
It is trained on the popular **Maps Dataset**, showcasing how deep learning can perform **image-to-image translation** tasks.

---

## 🧠 Project Overview

Pix2Pix is a type of **Generative Adversarial Network (GAN)** that learns a mapping from input images to output images using a paired dataset.  
In this project, the model learns to generate a **Google Maps-like view** from corresponding **satellite imagery**.

**Model Components:**
- 🧩 **Generator (U-Net):** Encodes satellite images → decodes into map-style images  
- 🧩 **Discriminator (PatchGAN):** Judges whether the generated image looks real  
- ⚔️ **Adversarial Training:** Generator tries to fool the Discriminator
